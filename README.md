######---使用方法---



-------

####"hukai_shisu.py"の使い方(CLI)
温度 湿度の順で引数を２つ入力して、当スクリプトを実行することで不快指数を計算して表示します。

例：温度30度、湿度80%の場合<br>

	python hukai_shisu.py 30 80
